# PrimeTrade – Scalable REST API

A secure, scalable REST API with JWT Authentication, Role-Based Access Control, and a Next.js frontend dashboard.

## Tech Stack
- **Backend:** Node.js, Express.js, TypeScript
- **Database:** MongoDB + Mongoose
- **Auth:** JWT + bcrypt
- **Validation:** Zod
- **Docs:** Swagger UI
- **Frontend:** Next.js + Tailwind CSS
- **Logging:** Winston
